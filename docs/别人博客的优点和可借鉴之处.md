## 二、配色与视觉系统

### 2.1 CSS 变量体系（可复用）

博客使用了 **126+ 个 CSS 自定义属性**，通过 `data-theme="dark"` 切换实现日夜模式。核心变量结构：

```css
:root {
  /* 基础色 */
  --global-bg: #fff;
  --font-color: #4c4948;
  --card-bg: rgba(255, 255, 255, 0.7);
  
  /* 卡片阴影 */
  --card-box-shadow: 0 3px 8px 6px rgba(7, 17, 27, 0.09);
  --card-hover-box-shadow: 0 3px 8px 6px rgba(7, 17, 27, 0.2);
  
  /* 强调色 */
  --default-bg-color: #2679cc;
  --btn-hover-color: #ff7242;
  --hover-color: #3498db;
  
  /* 自定义主题变量 */
  --liushen-nav-bg: rgba(255, 255, 255, 0.8);
  --liushen-card-bg: #fff;
  --liushen-card-border: 1px solid #e3e8f7;
  --liushen-maskbg: rgba(255, 255, 255, 0.85);
  --liushen-ai-bg: conic-gradient(from 1.5708rad at 50% 50%, #d6b300 0%, #42a2ff 54%, #d6b300 100%);
}

[data-theme="dark"] {
  --global-bg: #181818;
  --font-color: #f7f7fa;
  --card-bg: rgba(29, 30, 34, 0.7);
  --liushen-card-bg: #1d1e22;
  --liushen-card-border: 1px solid #42444a;
}
```

**借鉴点**：
- 所有颜色都通过 CSS 变量定义，切换主题只需改 `data-theme` 属性
- 卡片背景使用 **半透明 + backdrop-filter** 实现毛玻璃效果
- 阴影使用 `rgba(7,17,27,0.09)` 这种低饱和度颜色，更自然

## 三、卡片系统（核心亮点）

### 3.1 通用卡片样式

所有卡片（文章卡片、侧边栏小工具、友链卡片）共享同一套基础样式：

```css
.card-widget,
.recent-post-item,
.flink-list-item {
  background: var(--card-bg);
  box-shadow: var(--card-box-shadow);
  transition: all 0.3s;
  border-radius: 12px;
}

.card-widget:hover,
.recent-post-item:hover {
  box-shadow: var(--card-hover-box-shadow);
}
```

**借鉴点**：
- **统一的圆角**：全部使用 `12px` 圆角，视觉一致
- **悬浮加深阴影**：hover 时阴影加深，产生"浮起"效果
- **过渡动画**：`transition: all 0.3s` 让交互更平滑

### 3.2 文章卡片（recent-post-item）

文章卡片包含：**封面图 + 标题 + 摘要 + 元信息（日期/分类/标签/评论数）**

**封面图悬停缩放效果**：

```css
.recent-post-item .post_cover .post-bg {
  width: 100%;
  height: 100%;
  transition: filter 375ms ease-in 0.2s, transform 0.6s;
}

.recent-post-item:hover .post_cover .post-bg {
  transform: scale(1.1);
}
```

**借鉴点**：
- 图片悬停时 **延迟 0.2s 后缓慢放大**（`filter 375ms ease-in 0.2s`）
- 放大比例 `1.1`，不过分夸张
- 容器设置 `overflow: hidden` 防止图片溢出
